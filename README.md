## Description

### PART 1. 프로그램 제공 기능
 - 방 기능
 - 회원가입/로그인 및 로그아웃 기능
 - 코인 사용하기 및 시작하기 기능
 - 잔여 코인, 회원 이름, 회원 아이디 조회 기능
 - 코인 충전 기능
 - 사용 중인 방은 다른 색상으로 구분해주는 기능
 - 회원관리(회원정보 조회, 갱신, 수정, 병합, 삭제)가 가능한 관리자 모드 기능
